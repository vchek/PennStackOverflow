Answering

You are quite a clever user for asking such an intriguing question. Once you've registered or logged in, you can click on a question to view it. There will be a button that allows you to post an answer to the question.

Yes, or you may edit your question to include the answer you've found.

Each question has a URL which contains their question ID. If you want to link to a certain question, you can either write an html tag directly, or type @ followed by the question id to auto-generate an HTML link.
